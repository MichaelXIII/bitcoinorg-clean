How does Bitcoin work?

OK, this is a question that often causes confusion. Here's a quick explanation!

You might not need to know all this

But firstly, let's remember that you don't need to understand all the details to use Bitcoin or even have confidence that it works. If you can't follow this explanation, you can take note that the system has been working for multiple years and has not been hacked.

Blockchain

The entire Bitcoin network relies on the blockchain. Which is a shared public transaction log, in chronological order. All confirmed transactions are included in the blockchain with no exception. And there is a reason for this, the blockchain is used to check the balance of a Bitcoin address during a transaction. The integrity of the blockchain is enforced with cryptography.

Transaction

A transaction is a transfer of value between Bitcoin addresses that gets included in the blockchain. Bitcoin wallets keep a secret piece of data called a private key for each Bitcoin address. Private keys are used to sign transactions, providing a mathematical proof that they come from the owner of the addresses. The signature also prevents the transaction from being altered by anybody once it has been issued. All transactions are broadcast between users and confirmed by the network in the following minutes, through a process called mining.

Mining

Mining is a distributed consensus system that is used to include and confirm waiting transactions in the blockchain. It enforces a chronological order in the blockchain, protects the neutrality of the network and stops different computers disagreeing on the state of the system. To be confirmed, transactions must be packed in a block that fits very strict cryptographic rules that will be verified by the network. These rules prevent any previous block from being modified without invalidating all following blocks. And it creates the equivalent of a competitive lottery that prevents any individual from replacing parts of the blockchain and thus rolling back their own spends.

Going down the rabbit hole

This is only a very short and concise summary of the system. If you want to get into the details, you can read the original paper that describes the systems design, and explore the Bitcoin wiki.
