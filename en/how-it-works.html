How Bitcoin works?

Oh oh, here is a question that mixed up many people on earth. Let's try to explain it.. again!

You might not need to know all this

But first, lets put in perspective that trying to understand a system is often the best way to end up more confused by technical details that most users don't need to know. If you are seeking for confidence, you should probably stick to the fact that Bitcoin works and has been questionned since many years.

Blockchain

The entire Bitcoin network relies on the blockchain. Which is a shared public transaction log, in chronological order. All confirmed transactions are included in the blockchain with no exception. And there is a reason for this, the blockchain is used to check the balance of a Bitcoin address during a transaction. The integrity of the blockchain is enforced by cryptography.

Transaction

A transaction is a transfer of value between Bitcoin addresses that is later included in the blockchain. Bitcoin softwares keep a secret piece of data called a private key for each Bitcoin address. Private keys are used to sign each transactions, providing a mathematical proof that they come from their owner. The signature also prevent the transaction to be altered by anybody once it has been issued. All transactions are broadcasted between users and confirmed by the network in the following minutes through mining.

Mining

Mining is a distributed consensus system that is used to include and confirm waiting transactions in the blockchain. It enforces a chronological order in the blockchain, protects the neutrality of the network and prevents the blockchain to differ between users. To be confirmed, transactions must be packed in a block that must fit very strict cryptographic rules that will be verified by the network. These rules prevent any previous block to be modified without breaking the logic of all following blocks. And they create the equivalence of a competitive lottery that prevents any individual to consecutively write on the blockchain. Which would otherwise allow individuals to reverse their own transactions and double spend Bitcoins.
